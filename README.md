# Bank App


## Features
- User account creation with unique username and account number
- Password hashing for security (Will be added in future)
- Balance inquiry, deposit, and withdrawal
- Fund transfer between accounts (On going)
- Data stored in JSON format for easy access and modification

- ## Usage
- Follow on-screen prompts to create a new account
- Log in with your credentials
- Choose options to check balance, deposit, withdraw, or transfer money

## Installation
1. Clone the repository: `git clone https://github.com/yourusername/bank-application.git`
2. Run the followings in the terminal or command prompt:
   ```bash
   mkdir build
   cd build`
   cmake ..`
   cmake --build.`
4. Run the program: `./bankApp`
